最近由于业务需求需要做一个控制音量的滑块按钮组件，由于 js 实现起来比较复杂，需要考虑 touch 和 click 之间的切换。所以决定使用 h5 新增的 input range 来实现，只要改掉其默认样式就好。

几番搜索之后找到了 css tricks 的一篇文章，非常不错，所以再此简单记录顺带翻译(-_^)一下。（准不准确不好说）
